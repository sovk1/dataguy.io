PD Model Credit risk analysis

* Created a probability of Deafult model using Weight of Evidence (WoE), Information Value (IV) as metrics to predict the risk in lending.

Using Scikit, I employed Logistic regression by splitting the data set into 80/20 test/tain ratio. Additionaly performed p-values logistic regression. Tested the model with Gini, Kolmogorov-Smirnov co-efficients. Finally, created a scorecard based on credit score for easy inference of the model..
